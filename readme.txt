Introduction
Libft is a custom C library containing a collection of re-implemented standard C library functions and some additional utility functions. 
It provides essential tools for C programming and simplifies the development of C applications. The library is intended to be a learning resource as well as a practical tool for C developers.

Features
Re-implemented standard C library functions, such as printf, malloc, and strlen.
Additional utility functions for working with strings, linked lists, and more.
Well-documented and thoroughly tested code.
Easy to integrate into your C projects.
Helps reduce the need for repetitive coding when writing C programs.
